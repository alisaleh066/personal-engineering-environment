# Personal Toolkit

A comprehensive multi-domain toolkit combining mechanical, electrical, control systems, forex trading, and productivity tools.

## Features

### Engineering Calculators

#### Mechanical Calculator
- Beam Deflection (point/uniform loads)
- Spring-Mass-Damper analysis
- Simple Pendulum simulation
- Material presets (Steel, Aluminum, Titanium, etc.)
- Unit converter

#### Electrical Calculator
- RLC Transient Response
- RC Charge Curves
- Buck Converter Designer
- Component presets
- Ohm's Law calculator

#### Control Systems Suite
- **Second-Order System Designer**: Design from ζ/ωn or specs
- **Step Response Calculator**: Rise time, peak time, settling time, overshoot
- **PID Tuner**: Ziegler-Nichols method with config saving
- **Equivalence Bridge**: Convert mechanical/electrical to control parameters ⭐

### Trading Tools

#### Forex Suite
- Trade Calculator (position size, pip value, margin)
- Risk Analyzer (R:R ratio, trade quality rating)
- Trade Journal (log trades, track win rate, export)
- Quick Reference (lot sizes, pip values, rules)

### Productivity Suite
- **Pomodoro Timer**: Visual countdown with streak tracking
- **Assignment Tracker**: Deadlines, progress bars, priority
- **Grade Calculator**: Weighted grades, GPA, "what do I need?"
- **Study Analytics**: Sessions, hours, streaks, CSV export

### Data & Reference

#### Formula Library
- 54 built-in formulas (Electrical, Mechanical, Control, Forex, Statistics, Physics)
- Search functionality
- Browse by category
- Add custom formulas

#### CSV Analyzer
- Load any CSV file
- Statistical analysis (mean, median, std dev, min, max)
- Outlier detection
- Export results

#### Data Analysis Lab
- Store up to 3 simulations
- ASCII terminal plotting
- Multi-series comparison

### Utilities
- **Settings**: Colors, defaults, output directory
- **Help**: Context-sensitive guides per module
- **Self-Test**: System diagnostics

## Installation

### Prerequisites
- GCC compiler
- Python 3.x with matplotlib

```bash

./compile.sh
```

### Run
```bash
./apex
```

## Key Insights

The toolkit demonstrates mathematical equivalence across engineering domains:
- Spring-mass-damper ↔ RLC circuits ↔ Control systems
- All follow second-order differential equations
- Same damping ratio (ζ) = identical dynamic behavior

## Unit 1 Compliance

All C code uses only:
- Basic data types, operators, conditionals
- For/while loops, arrays, strings
- Structs, functions, pointers
- File I/O (fopen, fprintf, fscanf, fclose)

No dynamic memory allocation, linked lists, or recursion.
